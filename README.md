Sistema de Inventario para linea de peluches – Farmacia Los Berti

Este proyecto corresponde al desarrollo de un Sistema de Gestión de Inventario para la línea de peluchesde la Farmacia Los Berti.  
Incluye el diseño de la base de datos, triggers, auditoría y control de inventario.

Ramas del Proyecto:

Cada programador trabaja en su propia rama:

- `main` → Rama principal del proyecto  
- `Vegas` → Rama de Jose Vegas.
- `Berti` → Rama de Jose Berti.
- `Josue`  → Rama de Josue Pineda.
- `Sierra`  → Rama de Juan Sierra.
- `Jaider`  → Rama del Profesor Jaider Ponce.

El flujo de trabajo consiste en:

1. Cada programador trabaja en su rama.
2. Cuando completa una parte, crea un **Pull Request** hacia `main`.
3. El profesor revisa y aprueba los cambios.

Archivo principal: 
Incluye:

- Tablas principales (Peluches, Proveedores, Usuarios, Clientes, Entradas, Ventas…)
- Triggers de inventario (entradas y ventas).
- Triggers de auditoría.
- Restricciones de integridad.
- Datos iniciales para visualizar las relaciones.
